# Arduino UNO Sensor Shield PCB

An **Arduino UNO Compatible Sensor Shield** designed using **KiCad 10**. This PCB expands the Arduino UNO by providing dedicated Digital, Analog, Power, and Ground headers while integrating useful onboard components such as a buzzer, potentiometer, LEDs, and push buttons for rapid prototyping and embedded system development.

---

# Features

- Arduino UNO Compatible
- Digital I/O Expansion Headers
- Analog Input Headers
- Dedicated 5V & GND Headers
- On-board Buzzer
- 10K Potentiometer
- Power Indicator LED
- Two User LEDs
- Two Push Buttons
- Easy Sensor Connections
- Compact 2-Layer PCB
- Designed in KiCad 10
